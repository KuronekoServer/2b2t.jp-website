Japanese

2b2t Japanは日本で最も古いアナーキーサーバーでありルールやマップリセットもありません。
このサーバーはルールがありません。
サーバー内でできることが自由に行えます。
このサーバーの対応バージョンは1.12.2～1.18.1 です。
最新バージョンまでサポートしているので、バージョン変更は必要ありません。
このサーバーのワールドは2種類あり、1.12.2 と 1.18.1があります。

English

-You are free to do what you can within the server.
-Versions 1.12.2 to 1.18.1
-It supports the latest version, so there is no need to change the version.
-There is also a newer version of the world, so if you want to play the newer version, you can use 1.18.1.
-If you want to play the older version, you can use 1.12.2.
